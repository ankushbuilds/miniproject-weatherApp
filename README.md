# Weather App

A simple weather application built using **JavaScript**, **HTML**, and **CSS**, using the **OpenWeatherMap API**.

## Features

- Search for weather by city.
- Shows current temperature, min/max temperatures, and feels-like temperature.
- Displays weather description, humidity, and wind speed.
- Dynamic background images based on current weather.
- Shows current date and time for the selected city.
- Responsive design for desktop and mobile.

## Technologies Used

- JavaScript
- HTML5
- CSS
- OpenWeatherMap API
